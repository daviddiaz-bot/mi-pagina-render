Install.ps1

Script de configuración de discos para DSC (Desired State Configuration). Formatea y configura múltiples discos del sistema.

ℹ️
Información

Este script utiliza el formato DSC para definir recursos de configuración de discos.

Funcionalidades

  • Format-Disk-L: Formatea el disco 1 como unidad L (NTFS, GPT)
  • Format-Disk-S: Formatea el disco 2 como unidad S (NTFS, GPT)
  • Format-Disk-J: Formatea los discos 3 y 4 como unidades D y J (NTFS, GPT)
Script Format-Disk-L { TestScript = { $disk = Get-Disk -Number 1 $isValid = $false if ($null -ne $disk -and $disk.OperationalStatus -eq 'Online') { $partitions = Get-Partition -DiskNumber 1 $partitionCount = @($partitions) | Measure-Object | Select-Object -ExpandProperty Count if ($partitionCount -ge 1) { $isValid = $true } } return $isValid -eq $true } SetScript = { Initialize-Disk -Number 1 -PartitionStyle GPT New-Partition -DiskNumber 1 -UseMaximumSize -DriveLetter L Format-Volume -FileSystem NTFS -DriveLetter L } GetScript = { # Nothing } }

Uso

# Ejecutar el script .\Install.ps1 # O importar como módulo DSC Import-DscResource -ModuleName PSDesiredStateConfiguration

Script.ps1

Script de conexión remota mediante PowerShell Remoting (WinRM). Establece una sesión con una máquina remota y ejecuta comandos.

Funcionalidades

  • Conexión remota usando credenciales
  • Ejecución de comandos en la máquina remota
  • Gestión de sesiones PowerShell
# Definir las variables $targetMachine = "10.10.71.108" $username = "xxxxxxx" $password = "xxxxxx" # Convertir las credenciales en un objeto SecureString $securePassword = ConvertTo-SecureString -String $password -AsPlainText -Force $credentials = New-Object -TypeName System.Management.Automation.PSCredential -ArgumentList $username, $securePassword # Establecer la conexión try { $session = New-PSSession -ComputerName $targetMachine -Credential $credentials Write-Output "Conexión establecida exitosamente con $targetMachine." # Ejecutar comandos Invoke-Command -Session $session -ScriptBlock { Get-Process | Select-Object -First 5 } # Cerrar la sesión Remove-PSSession -Session $session } catch { Write-Error "Error al conectar con $targetMachine. Detalles: $_" }

Requisitos

  • WinRM habilitado en la máquina remota
  • Credenciales válidas con permisos de administrador
  • Conectividad de red entre máquinas

PostUpdateScript.ps1

Script de monitoreo post-actualización para servidores HP. Verifica el estado de actualización del servidor HP Storage.

Funcionalidades

  • Monitorea el registro HKLM:\Software\Hewlett-Packard\StorageServer
  • Verifica el valor HPIMFinalStatus
  • Espera hasta que la actualización se complete (status 0 o 1)
While($true) { Try { $reg = Get-ItemProperty -Path HKLM:\Software\Hewlett-Packard\StorageServer -Name "HPIMFinalStatus" if ($reg.HPIMFinalStatus = 0) { break; } elseif ($reg.HPIMFinalStatus = 1) { break; } else { continue; } } Catch { $ErrorMessage = $_.Exception.Message $FailedItem = $_.Exception.ItemName continue; } }
ℹ️
Valores de HPIMFinalStatus
  • 0: Actualización completada exitosamente
  • 1: Actualización completada con advertencias
  • Otro: Actualización en progreso

Depurar_Mevi_Files.bat

Script Batch para limpieza de archivos antiguos en el sistema MEVI.

Funcionalidades

  • Elimina archivos con más de 687 días de antigüedad
  • Busca recursivamente en D:\RPL_NAS_MED\MEVI\files
  • Usa el comando ForFiles para filtrar por fecha
@echo off Forfiles /p "D:\RPL_NAS_MED\MEVI\files" /s /m *.* -d -687 /c "cmd /c del /q @file" end

Parámetros

Parámetro Descripción
/p Ruta de búsqueda
/s Búsqueda recursiva
/m Máscara de archivo (*.* = todos)
/d Días de antigüedad (-687)
/c Comando a ejecutar

CopiaInventario.ps1

Script para respaldo automatizado de inventarios de VMware vCenter.

Funcionalidades

  • Copia el archivo consolidado de inventario
  • Crea carpetas con fecha para organización
  • Guarda en ubicación de respaldo
# Obtiene la fecha actual $fecha = Get-Date -Format "yyyyMMdd" # Ruta del archivo que deseas copiar $rutaArchivoOrigen = "\\servient\compartita-iyc\Documentos-IYC\Inventario\InventarioOnline\vCenterConsolidados.xlsx" # Ruta donde deseas guardar la copia del archivo $rutaCarpetaDestino = "\\servient\compartita-iyc\Documentos-IYC\Inventario\RVTOOLS\$fecha" # Si la carpeta destino no existe, la crea if (-not (Test-Path -Path $rutaCarpetaDestino)) { New-Item -ItemType Directory -Path $rutaCarpetaDestino | Out-Null } # Copia el archivo a la carpeta destino Copy-Item -Path $rutaArchivoOrigen -Destination $rutaCarpetaDestino Write-Host "El archivo ha sido copiado a la siguiente ubicación: $rutaCarpetaDestino"

Uso Programado

Se recomienda configurar este script como tarea programada para ejecución diaria o semanal.

ActualizarWin10.ps1

Script para automatizar la actualización de Windows 10 a la última versión.

Funcionalidades

  • Descarga el asistente de actualización de Windows 10
  • Ejecuta la actualización en modo silencioso
  • Guarda logs en directorio especificado
$d = "c:\win10exe" mkdir -Force $d # Crear el directorio si no existe $ComObj = New-Object System.Net.WebClient # URL de descarga del ejecutable de actualización de Windows 10 $exedl = "https://go.microsoft.com/fwlink/?LinkID=799445" # Ruta completa del archivo ejecutable de actualización $exe = "$d\Win10Upgrade.exe" # Descargar el archivo de actualización $ComObj.DownloadFile($exedl, $exe) # Verificar si la descarga fue exitosa antes de continuar if (Test-Path $exe) { Write-Host "Descarga del archivo completada correctamente." } else { Write-Host "Error: no se pudo descargar el archivo correctamente." exit 1 } # Iniciar el proceso de instalación con argumentos específicos Start-Process -FilePath $exe -ArgumentList "/quiet", "/skipeula", "/auto", "upgrade", "/dynamicupdate", "enable", "/copylogs", "$d"

Parámetros de Instalación

Parámetro Descripción
/quiet Modo silencioso sin interacción del usuario
/skipeula Omite el acuerdo de licencia
/auto upgrade Actualización automática
/dynamicupdate enable Habilita actualizaciones dinámicas
/copylogs Copia logs al directorio especificado
⚠️
Advertencia

Este script requiere reinicio del sistema. Ejecutar en ventana de mantenimiento programado.

Limpiar.bat

Script Batch para limpieza masiva de archivos antiguos en la unidad D.

Funcionalidades

  • Elimina archivos con más de 7200 días (~20 años)
  • Búsqueda recursiva en toda la unidad D
  • Eliminación silenciosa sin confirmación
@echo off ForFiles /p "D:\" /s /d -7200 /c "cmd /c del /Q @file" end
⚠️
Peligro

Este script elimina archivos permanentemente. Usar con precaución y verificar la ruta antes de ejecutar.

Scripts de endurecimiento y auditoría de PowerShell

AsegurarPS.cmd

Script completo de endurecimiento de PowerShell para Windows. Implementa múltiples capas de seguridad.

ℹ️
Objetivos del Script
  • Habilitar auditoría de PowerShell
  • Remover PowerShell 2.0
  • Configurar política de ejecución RemoteSigned
  • Limitar permisos en ejecutables PowerShell
  • Aplicar reglas de restricción de rutas

Configuraciones Aplicadas

Auditoría de PowerShell ▼

Habilita logging de módulos, bloques de script y transcripción:

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ModuleLogging" /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ModuleLogging" /v EnableModuleLogging /t REG_DWORD /d 1 /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ScriptBlockLogging" /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ScriptBlockLogging" /v EnableScriptBlockInvocationLogging /t REG_DWORD /d 1 /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\Transcription" /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\Transcription" /v EnableTranscripting /t REG_DWORD /d 1 /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell\Transcription" /v OutputDirectory /t REG_SZ /d "C:\Program Files\PSLogs" /f
Remoción de PowerShell 2.0 ▼
DISM /online /disable-feature /featurename:MicrosoftWindowsPowerShellV2Root /quiet DISM /online /disable-feature /featurename:MicrosoftWindowsPowerShellV2 /quiet
Política de Ejecución ▼
reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\PowerShell\1\ShellIds\Microsoft.PowerShell" /v ExecutionPolicy /t REG_SZ /d RemoteSigned /f reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\PowerShell" /v ExecutionPolicy /t REG_SZ /d RemoteSigned /f
Restricción de Permisos ▼

Limita el acceso a los ejecutables de PowerShell:

takeown /F C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe icacls C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe /inheritance:r /grant :r SYSTEM:M /C /Q /grant :r servient\userdp:F /C /Q

Revisarlogs.ps1

Script para analizar logs de ejecución de AsegurarPS.cmd y generar reportes de cobertura.

Funcionalidades

  • Procesa archivos de log generados por AsegurarPS.cmd
  • Identifica equipos cubiertos y faltantes
  • Genera reportes en carpeta de salida
  • Calcula estadísticas de ejecución
$folderlog = ".\" $ArchivosLog = Get-ChildItem "$folderlog\*log*.txt" $TotalCubiertos = 0 $TotalFaltantes = 0 $Fecha = Get-Date -Format "ddMMyyyy" Clear-Host if ((Test-Path -Path .\Salida) -eq $false){New-Item -ItemType Directory -Path .\Salida} foreach ($log in $ArchivosLog) { Write-Host "`nProcesando archivo log '$($log.Name)'" -ForegroundColor White $Info = Get-Content -Path $log.FullName $EquiposCubiertos = $Info | Select-String "process ID" foreach ($linea in $EquiposCubiertos){ $Cubiertos += ($linea -replace '(.+?) with.+','$1').Replace("AsegurarPS.cmd started on ","") } $Cubiertos = $Cubiertos | select -Unique $EquiposFaltantes = $Info | Select-String "Couldn't access" foreach ($linea in $EquiposFaltantes ) { $valor = (( ($linea -replace '(.+?)\.\.\. .+','$1').Replace("Couldn't access ","")).replace(":","")).replace("Connecting to ","") if ($valor -notin $Cubiertos){$Faltantes += $valor} } $Faltantes = $Faltantes | select -Unique $Total = $Cubiertos.Count + $Faltantes.count $TotalCubiertos += $Cubiertos.Count $TotalFaltantes += $Faltantes.count } Write-Output " El total de equipos cubiertos es de $TotalCubiertos" Write-Output " El total de equipos faltantes es de $TotalFaltantes"

Archivos Generados

  • Resumen-[fecha].txt - Resumen general
  • [log]-ListaEquiposCubiertos-[fecha].txt - Equipos procesados
  • [log]-ListaEquiposFaltantes-[fecha].txt - Equipos pendientes
